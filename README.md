# Свадебный лендинг

Откройте `index.html` в браузере — сборка и сервер не нужны.

## Как редактировать

- Тексты, имена, адрес и расписание: `index.html`.
- Дата обратного отсчёта: первая строка `script.js`.
- Цвета: переменные в начале `styles.css`.
- Фотографии: замените `assets/wedding-photo-set.png` картинкой-коллажем 2×2 либо пропишите отдельные изображения в классах `.photo-1` — `.photo-4`.
- Ссылка на карту: найдите `https://maps.google.com` в `index.html`.

Форма сейчас сохраняет ответ в браузере гостя. Для реального сбора ответов вставьте ссылку Google Forms в кнопку или подключите Formspree/Telegram-бота.

## GitHub Pages

В настройках репозитория: **Settings → Pages → Deploy from a branch → main / root → Save**.
